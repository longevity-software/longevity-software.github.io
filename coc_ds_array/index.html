C Arrays

An Array is the simplest data structure in C and most other languages, it is a block of continuous memory which is reserved for storing values of the same type.

The key features of an array are:

  • Each element in the array is stored directly after the previous element in memory
  • Array indexes start at 0
  • Each element in the array must be the same type e.g. an array of integers
  • Arrays form the basis of many of the more complicated data structures
